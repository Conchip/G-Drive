두번째 챕터

김철수는 복권을 샀다

git log : 커밋 기록을 볼 수 있다

git reset --hard HEAD^ : 커밋 전 단계로 돌아간다

git checkout 커밋코드 : 커밋 시점으로 돌아간다

mixed reset : 커밋만 삭제하고 변경사항은 그대로 둔다.

git reset HEAD^

■ 복합 리셋
▷ 커밋만 삭제하고 파일 변경 사항은 그대로 둔다.
▷ 변경 사항들을 unstage 영역에 둔다.

하드 리셋 - 파일 변경 내역을 유지하지 않는다.
복합 리셋 - 변경한 파일의 내용은 그대로 둔다.

■ 원격저장소 확인
git remote -v
■ origin에서 코드가져오기
git pull origin master
